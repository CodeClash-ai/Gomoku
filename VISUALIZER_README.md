## Installation

Flask is required to run the visualizer:

```bash
uv pip install flask --system
```

## Usage

### Running the Visualizer

```bash
python visualizer.py
```

This will start a web server at `http://localhost:5000`. Open this URL in your browser to view the visualizer.

### Options

```bash
python visualizer.py --port 8080              # Run on a different port
python visualizer.py --logs-dir my_logs       # Use a different logs directory
```

### Generating Game Logs

The visualizer works with JSON log files generated by the game engine. The engine has been updated to automatically create both `.log` (text) and `.json` files.

To generate games with logs:

```bash
python engine.py main.py main.py -r 10 -o logs
```

This will create game logs in the `logs/` directory.

Use the speed slider to adjust playback speed from 0.5x to 3.0x.

## Log Format

The visualizer reads JSON files with the following structure:

```json
{
  "game_number": 1,
  "timestamp": "2025-12-18 15:40:25",
  "board_size": 15,
  "players": {
    "black": {
      "name": "player1",
      "path": "main.py"
    },
    "white": {
      "name": "player2",
      "path": "main.py"
    }
  },
  "winner": "player2",
  "error": null,
  "moves": [
    {"move_number": 1, "player": "black", "x": 7, "y": 7},
    {"move_number": 2, "player": "white", "x": 7, "y": 8}
  ]
}
```

## Troubleshooting

**No games showing up?**
- Make sure you've run games with the `-o logs` option
- Check that the logs directory contains `.json` files
- Verify the `--logs-dir` path is correct

**Server won't start?**
- Check if Flask is installed: `python -c "import flask"`
- Try a different port: `python visualizer.py --port 8080`
- Make sure no other service is using port 5000
